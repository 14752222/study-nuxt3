全局样式 

 - 方法1
在nuxt.config.js中配置
```js
export default defineNuxtConfig({
    devtools: { enabled: true } ,
    css: [
        'assets/css/main.css'
    ],
})
```
 - 方法2
在assets/css/下新建一个main.css文件
，然后在app.vue中引入
```vue
<script>
import '~/assets/css/main.css'
</script>

```
 - 方法3
在app.vue中直接写
```vue
<style>
body{
    background-color: #f00;
}
</style>
```

## 2.全局样式的优先级
 - 方法1 > 方法2 > 方法3


# 全局变量
在css文件夹中创建一个variables.css文件
```css
:root{
    --color: #f00;
}
```

在nuxt.config.js中配置
```js
export default defineNuxtConfig({
    devtools: { enabled: true } ,
    vite: {
        css: {
            preprocessorOptions: {
                scss: {
                    additionalData: `@import "@/assets/css/variables.css";`
                }
            }
        }
    },
})
```
